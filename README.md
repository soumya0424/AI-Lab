## Overview

AI-Lab is a structured set of Artificial Intelligence lab notebooks covering supervised learning, model evaluation, and core ML algorithms. Each lab builds on the previous to reinforce foundational AI concepts through practice.

### Why AI-Lab?

This repository provides a lab-by-lab learning path through AI fundamentals. The core topics include:

- 🤖📚 **Supervised Learning**: Classification and regression across multiple algorithms.
- 📊🔬 **Model Evaluation**: Accuracy, precision, recall, and cross-validation techniques.
- 🧩📌 **Algorithm Coverage**: From linear models to ensemble methods across 8 labs.
- 📸📊 **Visual Analysis**: Screenshots and plots documenting learning outcomes.
- 📈📦 **Progressive Complexity**: Labs 1 through 8 with increasing difficulty.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language**: Python 3.8+
- **Packages**: `scikit-learn`, `pandas`, `numpy`, `matplotlib`

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/soumya0424/AI-Lab
   cd AI-Lab
   ```

2. **Install dependencies:**

   ```bash
   pip install scikit-learn pandas numpy matplotlib jupyter
   ```

### Usage

```bash
jupyter notebook
```

Open labs in order: `Lab1.ipynb` → `Lab2.ipynb` → ... → `Lab 8.ipynb`
